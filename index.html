Here are some of my latest projects completed while at General Assembly.

MUSICAL SQUARES
This project focused on delivering a Minimum Viable Product (MVP) within 3 days using JavaScript /jQuery. I created a game that manages the interaction of randomised audio clips and images, with timers to enable the fading of images in response to user inputs. The web page was created using HTML5/Responsive CSS3 for support of mobile/tablet Front End interaction with a Back End served with PHP.
Build: JavaScript/ jQuery, HTML5, Responsive CSS3, CLI, PHP, Git, Heroku, Trello.

CONQUIZTADORS
A collaborative team project developing a RESTful game in which a player takes on the role of a Marvel superhero. The solution makes use of external APIs from Marvel, Google, and RestCountries.eu. An internal API with country details was also used. The database uses a NoSQL MongoDB with a NodeJS server on the Back End. I developed the quiz logic which drove user inputs for the game and win outcomes.
Build: Express, HTML5, SCSS, NodeJS, MongoDB, MVC, external APIs. Git, NPM

TWITTER4LONDON
This is a RESTful, MEAN stack solution developed by a team of four. Users interact with a mobile interface, consuming external APIs from Twitter and Transport for London, Underground; providing realtime updates from users. The app authenticates with OAuth for Facebook, GitHub and Twitter so users can easily sign-in. Users are able to filter by line and post line specific messages for other users of the app.
Build: AngularJS, NodeJS, MongoDB, TfL and Twitter APIs, JWT/OAuth, Git, NPM.

SOBUYLOCAL
This is a RESTful desktop and mobile app I developed using an AngularJS Front End and Ruby /Ruby on Rails Back End. Postgres SQL was used to manage an internal API created for customer/ store and product details. Insomnia was used for Back End resource management, while Bootstrap enabled the rapid deployment of styling updates. I used Trello, Balsamiq and MoSCoW to support planning and delivery of key tasks to achieve MVP and deployment of the solution via GitHub and Heroku.
Build: AngularJS, Ruby on Rails, SQL/Postgres, JWT, Bootstrap/CSS3/HTML5. Gulp.
